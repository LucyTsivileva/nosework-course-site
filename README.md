# Nosework Course Site

Новый управляемый проект для переноса сайта `nosework.lucydogs.pro` с Tilda на `Cloudflare Pages`.

## Что здесь лежит

- `docs/` — документация по проекту, памяти, доступам и шагам публикации
- `functions/` — серверные функции для форм на Cloudflare Pages
- `public/images/` — локальные изображения и исходные материалы
- `snapshots/` — снимки текущих страниц и экспортированные текстовые материалы
- `src/components/` — секции лендинга как независимые модули
- `src/content/` — длинные тексты и вспомогательные заметки
- `src/data/` — структурированный контент сайта

## Текущий статус

- Сохранён HTML-снимок действующего сайта с Tilda
- Сохранён HTML-снимок страницы `lucydogs.pro/price` как стилистический референс
- Скопированы локальные изображения в `public/images/source-assets/`
- Создан каркас нового сайта и заготовка формы для Cloudflare
